# Dropdown Menu

Dropdown Menu is a non-modal dialog that floats around a trigger. It's used to display contextual information to the user, and should be paired with a clickable trigger element.

## Default

<Preview withToolbar >
    <Story name="Default">
      <DropdownMenu
        trigger={props => <Button {...props}>Click me</Button>}
        items={[
          <Link variant="menu">Hello</Link>,
          <Link variant="menu">Hi</Link>
        ]}
      />
    </Story>
</Preview >
